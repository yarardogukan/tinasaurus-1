# Overview

Welcome to our documentation, where you can find information, best practices, and guidance on our **QBot RPA** solution.

## Getting Started

Start by creating a new project. The QBot RPA Studio login screen is as follows. You must enter your user information to be able to connect.

![Alt Text](../videos/QbotRpaStudio-Open.gif)
