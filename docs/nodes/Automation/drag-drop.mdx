> Moves a specified object to another location. It takes two parameters, source and target.

## Parameters

> **Select Coordinate** : It automatically selects the source and target coordinate values.
>
> **Source Snap Path** : Automatically takes and attaches the snap of the file to be moved.
>
> **Target Snap Path** : Automatically takes and attaches the snap of the area where the file will be moved.
>
