> Scroll automates scrolling on the screen.

## Parameters

> **Direction** : Specifies the scrolling direction. **Type** : Up/Down/Left/Right
>
> **Line** : Determines how far to scroll numerically on the screen. **Type** : Integer
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
>
