> The CreateExcel component is used to create an excel file.

## Parameters

> **Force to create** : If you check the 'ForcetoCreate' box, it will overwrite even if excel has that name you want to create.
>
> **Folder Path** : When you click the button, you must choose the place to save on the screen that appears in front of you. **Type** : String
>
> **File Name** : You must write the name of the file you want to save. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
