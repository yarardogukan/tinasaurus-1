# Csv to Excel

> The csvtoexcel component converts csv files to excel files.

>> # Parameters

>>> **Folder Path** : When the button is clicked, you must select the csv file you want to convert. **Type** : String

>>> **File Name** : When you click the button, you must choose the place to save on the screen that appears in front of you. **Type** : String

>>> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
