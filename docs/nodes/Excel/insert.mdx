# Insert

> The insert component allows you to enter data into a specific cell in Excel.

>> # Parameters

>>> **Design Type** : You should choose the Excel entry design.

>>> **Desing** : If you selected desing type you should enter the design. **Type**: String

>>> **Sheet Name** : Enter the Sheet Name you want to work with. **Type**: String

>>> **Text** : You should enter the text for the insert excel cell. **Type** : String

>>> **Row Number** : You need to enter the row number you want to add. **Type** : Integer

>>> **Column Number** : You need to enter the column number you want to add. **Type** : Integer

>>> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
