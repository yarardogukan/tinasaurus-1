# Read Excel

> The ReadExcel component helps us to read the data in the excel file.

>> # Parameters

>>> **Is Header** : When you click on the Is Header checkbox, it assigns the first column of excel as the excel header. **Type** : String

>>> **Start Row** : You specify the starting row of excel to read. **Type** : Integer

>>> **End Row** : You specify the ending row of excel to read. **Type** : Integer

>>> **Start Column** : You specify the starting column of excel to read. **Type** : Integer

>>> **Column Name** : You need to write the column name you want to read. **Type** : String

>>> **Sheet Name** : Enter the Sheet Name you want to work with. **Type**: String
