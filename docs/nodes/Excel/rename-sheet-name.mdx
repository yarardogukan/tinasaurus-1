> The RenameSheetName component helps us to change to sheetname the excel file.

## Parameters

> **Sheet Name** : Insert the old sheet name. **Type** : String
>
> **New Sheet Name** : Insert the new sheet name. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
