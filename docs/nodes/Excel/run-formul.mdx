# Run Formula

> The RunFormul component allows you to run a formula in a cell you want in excel.

>> # Parameters

>>> **Row Number** : Enter the row number you want the formula to run. **Type** : Integer

>>> **Column Number** : Enter the column number you want the formula to run. **Type** : Integer

>>> **Formula** : Enter the formula you want to run. **Type** : String

>>> **Sheet Name** : Enter the Sheet Name you want to work with. **Type**: String

>>> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
