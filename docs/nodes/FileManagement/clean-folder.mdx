> The cleanfolder component is used to delete all file in folder.

## Parameters

> **Folder Path** : When you click the button, you must choose the place to delete on the screen that appears in front of you. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
