> The copyfile component is used to copy a file and paste a file into a folder.

## Parameters

> **Source File Path** : Click the button and select the file you want to copy. **Type** : String
>
> **Target Folder Path** : Click the button and choose where you want to save the copied file. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
