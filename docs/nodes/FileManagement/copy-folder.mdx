# Copy Folder

> The Copyfolder component is used to copy and paste a folder into another folder.

> > # Parameters

> > > **Source Folder Path** : Click the button and select the folder you want to copy. **Type** : String

> > > **Target Folder Path** : Click the button and choose where you want to save the copied folder. **Type** : String

> > > **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
