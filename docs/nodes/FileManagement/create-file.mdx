> The CreateFile component is used to create a file.

## Parameters

> **File Path** : Click the button and select the folder where you want to create the file. **Type** : String
>
> **File Name** : Enter the name of the file you want to create with its extension. **Type** : String
>
> **Result** : Returns the full file path. **Type** : String
