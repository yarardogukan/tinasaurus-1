# Create Folder

> The CreateFolder component is used to create a folder.

> > # Parameters

> > > **File Path** : Click the button and select the folder where you want to create the folder. **Type** : String

> > > **File Name** : Enter the name of the folder you want to create. **Type** : String

> > > **Result** : Returns the full folder path. **Type** : String
