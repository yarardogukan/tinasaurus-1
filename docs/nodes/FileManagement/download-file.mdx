> The DownloadFile component allows you to download files from a url.

## Parameters

> **URL** : You need to enter the URL where the file is located. **Type** : String
>
> **File Path** : Click the button and choose where you want to save the downloaded file. **Type** : String
>
> **File Name** : Enter the name of the file you want to create with its extension. **Type** : String
>
> **Result** : Returns the full file path. **Type** : String
