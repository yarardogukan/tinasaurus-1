> The MoveFile component move to file another folder.

## Parameters

> **Source File Path** : Click the button and select the file to move it to another folder. **Type** : String
>
> **Target File Path** : Click the button and select the folder you want moved. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
