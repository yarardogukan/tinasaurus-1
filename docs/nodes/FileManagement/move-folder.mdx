# Move Folder

> The MoveFolder component move to folder another folder.

> > # Parameters

> > > **Source Folder Path** : Click the button and select the folder to move it to another folder. **Type** : String

> > > **Target Folder Path** : Click the button and select the folder you want moved. **Type** : String

> > > **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
