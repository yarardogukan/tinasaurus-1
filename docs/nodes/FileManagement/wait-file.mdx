> The WaitFile component waits file in the folder.

## Parameters

> **File Path** : Click the button and select the file for waiting. **Type** : String
>
> **Timeout** : Insert timeout second to wait file **Type** : Integer
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
