# Write File

> The WriteFile component allows you to write to the file .

> > # Parameters

> > > **Source File Path** : Click the button and select the file to Write text. **Type** : String

> > > **Write Text** : Write the text you want to write to the file. **Type** : String

> > > **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
