# SnapShot Image

> The SnapShotImage component is used to take a snapshot of a specific area on the screen. This component is a NativeActivity and It creates a session that includes your image file's information to share with other 

>> # Parameters

>>> **Image Path** : When we press the screenshot button, the screen will open to take a screenshot. When we take a screenshot from here, it fills automatically. **Type** : String
