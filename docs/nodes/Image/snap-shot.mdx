> The SnapShot component allows the process to take a screenshot of the area determined during operation.

## Parameters

> When the **select coordinate** button is clicked, it allows us to select the desired area on the screen and fill the parameters or you can fill in the parameters manually.
>
> **File Path** : You need to enter the location where you want to save the file and the name of the file with extension. **Type** : String
>
> **Start Coordinate** : You must enter the starting point upper left. **Type** : Integer
>
> **End Coordinate** : You must enter the starting point bottom right. **Type** : Integer
