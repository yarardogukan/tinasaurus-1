> The WaitImage component helps us to waits for the picture to appear on the screen.

## Parameters

> **Time Out** : Insert timeout second to wait image. **Type** : Integer
>
> **Similarity** : You should enter how much similarity ratio you want, the default value is 0.85. **Type** : Single
>
> **Screeen** : If the number of monitor you use is more than 1, you can choose the monitor you will use here, the default value is 0. **Type** : Integer
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
