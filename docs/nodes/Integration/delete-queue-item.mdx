> The DeleteQueueItem component delete an item in queue.

## Parameters

> **Queues** : Select the queue. **Type** : String
>
> **Items** : Select the you want delete item in queue. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
