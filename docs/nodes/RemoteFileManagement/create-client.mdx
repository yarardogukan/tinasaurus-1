> The CreateClient component allows us to connect to remote desktop, and This is a NativeActivity and It creates a session that includes your connection's information to share with other components.

## Parameters

> **Client** : Select the client type. **Type** : String
>
> **Host** : You must enter the address of the machine you want to connect to  **Type** : String
>
> **User Name** : You must enter the username. **Type** : String
>
> **Password** : You must enter the password. **Type** : String
