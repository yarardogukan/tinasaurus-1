> The CreateFolder component creates a folder within a folder on the remote desktop.

## Parameters

> **Directory** : Insert to directory path to create folder. **Type** : String
>
> **Folder Name** : Insert to folder name **Type** : String
>
> **Result** : Returns the full path of folder. **Type** : String
