> The DeleteFile component helps us to delete file in remote desktop.

## Parameters

> **Directory** : Insert to directory path to delete file. **Type** : String
>
> **File Name** : Enter the file name you want to delete. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
