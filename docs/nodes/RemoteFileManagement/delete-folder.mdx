> The DeleteFolder component helps us to delete folder in remote desktop.

## Parameters

> **Directory** : Insert to directory path to delete folder. **Type** : String
>
> **File Name** : Enter the folder name you want to delete. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
