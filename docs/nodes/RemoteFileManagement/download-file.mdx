> The DownloadFile component helps us to download file in remote desktop to current desktop.

## Parameters

> **Remote File Patj** : Insert to directory path to download file. **Type** : String
>
> **Download File Directory** : Insert the folder path you want to save in your current desktop. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
