# Execute Command

> The ExecuteCommand component run command on the remote desktop.

>> # Parameters

>>> **Command** : Insert to command. **Type** : String

>>> **Output** : If the command is run successfully, the result will return here.  **Type** : String

>>> **Error** : If the command gets an error, it will return here. **Type** : String

>>> **Exit Code** : Returns exit code **Type** : Integer
