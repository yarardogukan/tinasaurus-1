> The ExistFile component checks if the file exists on remote desktop.

## Parameters

> **Directory** : Insert to directory path to checks folder. **Type** : String
>
> **File Name** : Insert to file name you want to check. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
