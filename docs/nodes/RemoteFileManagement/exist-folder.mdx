> The ExistFolder component checks if the folder exists on remote desktop.

## Parameters

> **Directory** : Insert to directory path to checks folder. **Type** : String
>
> **Folder Name** : Insert to folder name you want to check. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
