# PostgreSQL Connection

> The PostgreSQLConnection component is help us to connection to database. This component is a NativeActivity and It creates a session that includes your Database's information to share with other components.

>> # Parameters

>>> **Server** : Insert to database Server  **Type** : String

>>> **Database** : Insert to database name **Type**: String

>>> **User Name** : Insert to database user name **Type** : String

>>> **Password** : Insert to database user name **Type** : String
