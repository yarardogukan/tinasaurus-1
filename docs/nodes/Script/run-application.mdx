# Run Application

> The RunApplication component runs an application.

>> # Parameters

>>> **File Path** : Click the button and select the file you want to run. **Type** : String

>>> **Run as administrator** : runs the application as administrator.

>>> **Environments** :  You need to enter the dictinary you created before **Type** : Dictonary(string,string)

>>> **Output** : Returns the output of the application **Type** : String

>>> **Error** : If it gets an error, it returns the error here **Type** : String
