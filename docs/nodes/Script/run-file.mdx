# Run File

> The RunFile component runs an file example: python,vbs,javascript.

>> # Parameters

>>> **File Path** : Click the button and select the file you want to run. **Type** : String

>>> **Environments** :  You need to enter the dictinary you created before **Type** : Dictonary(string,string)

>>> **Result** : Returns the output of the application **Type** : String
