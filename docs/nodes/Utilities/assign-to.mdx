# AssignTo

> The AssignTo component ensures that the values ​​that are key and value reach their value when you give them the key

>> # Parameters

>>> **Item Name** : Enter the object with key and value  **Type** : String

>>> **Key** : Enter the key of the value for which you want to find the value **Type**: String

>>> **Result** : Returns the value **Type** : String
