# Date Time Format

> The DateTimeFormat component helps us to format date time.

>> # Parameters

>>> **Format** : Select the date time format. **Type** : String

>>> **Custom Format** : If select the custom format you can enter custom format. **Type** : String

>>> **Date** : Insert the date time **Type** : DateTime

>>> **Culture** : Insert the your culture (examp "tr-TR") **Type** : String

>>> **Result** : Returns the output of the operation. **Type** : String
