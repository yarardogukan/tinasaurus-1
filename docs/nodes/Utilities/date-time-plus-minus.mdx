# Date Time Plus Minus

> The DateTimePlusMinus component helps us to plus or minus date time.

>> # Parameters

>>> **Operations** : Select the operation. **Type** : String

>>> **Date** : Enter the date **Type** : DateTime

>>> **Value** : Enter the value for plus or minus **Type** : Integer

>>> **Result** : Returns the output of the operation. **Type** : DateTime
