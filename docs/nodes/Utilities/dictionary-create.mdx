> The DictonaryCreate component helps us to create dictionary. Enter Key and Value and Click on the add button this component has created a dictionary.

## Parameters

> **Key** : Enter the dictonary key. **Type** : String
>
> **Value** : Enter the dictonary value **Type** : String
>
> **Result** : Returns the output of the operation. **Type** : Dictonary(String,String)
