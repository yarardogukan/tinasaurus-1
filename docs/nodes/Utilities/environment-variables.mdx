# Environment Variables

> The EnvironmentVariables component helps us to get or set environment variable.

>> # Parameters

>>> **Operation** : Select the operation. **Type** : String

>>> **Variable Target** :  Select the target. **Type** : String

>>> **Environment** : Enter the dictonary for the environment **Type** : Dictionary(String,String)

>>> **Result** : Returns the output of the operation. **Type** : Dictionary(String,String)
