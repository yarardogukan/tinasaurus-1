# Increase Decrease

> The  IncreaseDecrease component helps us to Increase or Decrease number.

>> # Parameters

>>> **Operation** : Select the operation. **Type** : String

>>> **Number** : Enter the number you want to increase and decrease. **Type** : Integer

>>> **Value** : Enter the value you want to increase or decrease. **Type** Integer

>>> **Result** : Returns the output of the operation. **Type** : Integer
