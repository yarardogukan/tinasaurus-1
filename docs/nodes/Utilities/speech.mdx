# Speech

> The Speech component helps us to text to speech.

>> # Parameters

>>> **Voice** :  Select the voice type **Type** : String

>>> **Text** : Enter the text you want to voice **Type** : String

>>> **Rate** : Enter the speech rate. **Type** : Integer

>>> **Volume** : Enter the speech Volume. **Type** : Integer

>>> **Audio File Path** : Click the button and choose where you want to save the speech file. **Type** : String

>>> **Result** :  Returns the full path of the speech. **Type** : String
