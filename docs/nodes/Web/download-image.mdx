> The DownloadImage component is allows to download an image in website.

## Parameters

> **XPath** :  Enter xpath of area to download image. **Type** : String
>
> **Folder Path** : Enter the full path to where the downloaded image will be saved. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
