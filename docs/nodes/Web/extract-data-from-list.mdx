# Extract Data From List

> The ExtractDataFromList component is allows to read a table of the website.

>> # Parameters

>>> **XPath** :  Enter xpath of the table. **Type** : String

>>> **Environments** : Enter dictonary,  value should be the xpath of the cell. **Type** : Dictonary

>>>**Result** : Returns the JArray of the table data **Type** : JArray
