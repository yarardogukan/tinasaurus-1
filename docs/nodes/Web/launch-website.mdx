> The LaunchWebSite component is allows to open the website. This component is a NativeActivity and It creates a session that includes your web information to share with other components.

## Parameters

> **Driver** :  Select the webdriver. **Type** : String
>
> **Url** : Enter the url of the website. **Type** : String
