> The Select component is allows to select an item in the drag drop menu.

## Parameters

> **XPath** :  Enter the Xpath. **Type** : String
>
> **Select** : Enter the text of the item to be selected. **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
