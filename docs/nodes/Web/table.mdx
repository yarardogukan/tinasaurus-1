> The Table component allows to read and extract csv file.

## Parameters

> **XPath** :  Enter the xpath of the table. **Type** : String
>
> **File Path** : Enter the path of the file to be saved **Type** : String
>
> **Result** : Returns the status of performing the operation correctly. **Type** : Boolean
